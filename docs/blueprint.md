# **App Name**: PassDec

## Core Features:

- User Account & Profile Management: Users can sign up, log in, create, and manage their profile details including type (club, individual), city, description, and upload a profile picture.
- Offer Posting: Authenticated users can create new offers, specifying type (sell, swap, event, match), title, description, location (city/geoPoint), price or event date, and upload up to two photos.
- Offer Discovery & Filtering: Users can browse all active offers. A dedicated 'home screen' with distinct buttons allows filtering offers by type (e.g., selling, swapping, matching), and users can also filter by city.
- In-App Messaging & Conversations: Users can initiate and participate in private, text-only conversations related to specific offers, sending and receiving messages to other participants.
- Message Notifications: Users receive simple push notifications via Firebase Cloud Messaging when a new message arrives in their conversations.

## Style Guidelines:

- Color scheme: Dark theme, suggesting modernity and dynamic energy. Primary interactive color: Vibrant purple (#DB47F5), conveying a strong, active presence without being overly aggressive. Background color: A desaturated, deep indigo-purple (#2E2630), providing a rich, dark canvas that highlights content. Accent color: A luminous lavender-blue (#B5A1FE), analogous to the primary color, used to draw attention to important elements and actions.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and objective aesthetic, ensuring readability and a consistent digital experience across the application.
- Adopt clear, intuitive icons with a subtle athletic or game controller aesthetic, particularly for the main navigation buttons (e.g., cross, circle, triangle, square), enhancing visual cues for offer categories.
- Utilize a clean, card-based layout for offer listings, making information digestible. Conversation interfaces should be straightforward, prioritizing message display and input for ease of use.
- Incorporate subtle, responsive transitions for navigating between views and loading new content, providing a smooth and modern user experience without being distracting.
